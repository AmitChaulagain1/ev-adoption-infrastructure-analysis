# EV Adoption Growth & Infrastructure Planning Analysis

## Project Overview

This project analyzes electric vehicle (EV) adoption trends across Washington State to identify geographic concentration patterns and infrastructure planning needs.

The analysis focuses on:

* EV adoption growth trends
* County-level concentration
* BEV vs PHEV distribution
* Infrastructure and grid planning implications

## Technologies Used

* SQL
* Snowflake
* AWS S3
* Data Visualization
* Exploratory Data Analysis (EDA)
* ELT Workflow

## Dataset

Washington State EV Population Data (Data.gov)

## Key Insights

* EV adoption accelerated significantly after 2020
* Approximately 70% of recent EV growth came from three counties
* High-adoption counties were primarily dominated by fully electric vehicles (BEVs)
* Results highlight increasing localized pressure on charging infrastructure

## Project Workflow

1. Stored raw CSV data in Amazon S3
2. Loaded data into Snowflake
3. Processed and analyzed data using SQL
4. Performed exploratory data analysis
5. Visualized trends and infrastructure implications
